California Housing Machine Learning Project

Project Description

Այս project-ում մենք աշխատում ենք California Housing dataset-ի հետ։ Նպատակն է ուսումնասիրել տվյալները և հասկանալ բնակարանների գների վրա ազդող հիմնական գործոնները։

Project-ը ներառում է տվյալների ներբեռնում, ուսումնասիրում (EDA) և նախնական մշակման քայլեր։

⸻
Dataset

Օգտագործվում է California Housing dataset-ը, որը պարունակում է տեղեկություններ՝

* Միջին եկամուտ (median income)
* Տան միջին տարիքը
* Սենյակների քանակը
* Տան արժեքը (median house value)
* Այլ բնակարանային վիճակագրություն


Steps in Project

1. Libraries Import

Օգտագործվել են հիմնական գրադարաններ՝

* pandas
* numpy
* seaborn
* urllib


2. Data Download

Տվյալները ներբեռնվում են ինտերնետից և պահվում local folder-ում։


3. Data Loading

CSV ֆայլը կարդացվում է pandas-ի միջոցով։

⸻

4. Data Exploration (EDA)

Ուսումնասիրվում են տվյալների հիմնական հատկությունները՝

* տվյալների չափը
* բաց արժեքներ (missing values)
* բաշխումներ (distributions)
* correlation analysis

⸻

5. Visualization

Օգտագործվում են գրաֆիկներ տվյալների հասկանալու համար։


Goal

Հասկանալ, թե որ feature-ներն են ամենաշատը ազդում բնակարանի արժեքի վրա և պատրաստել տվյալները machine learning model-ի համար։

Possible Improvements
